# WPF Graphics Editor 🎨

A robust, vector-based drawing application built with **C#** and **WPF**. This project serves as a practical implementation of fundamental software design patterns to ensure scalability and maintainability.

## 🚀 Technical Highlights
* **Advanced Undo/Redo System:** Implemented using the **Command Pattern**, allowing seamless state management of user actions.
* **Modular Brush Management:** Utilizes the **Factory Method Pattern** for centralized and decoupled creation of drawing tools and colors.
* **Image Rendering:** Supports high-quality exports in **PNG, JPG, and BMP** formats using `RenderTargetBitmap`.
* **Dynamic UI:** Features adjustable stroke thickness, text overlays, and an intuitive toolset.

## 🏗️ Architecture & Patterns
This project demonstrates more than just UI; it focuses on clean code and architectural excellence:
* **Command Pattern:** Each drawing action is encapsulated as an object, making the history stack manageable.
* **Factory Pattern:** Simplifies the logic of resource creation, adhering to the Single Responsibility Principle.



## 🛠️ Tech Stack
* **Language:** C#
* **Framework:** .NET / WPF (Windows Presentation Foundation)
* **Design Patterns:** Command, Factory Method
